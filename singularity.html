Singularity


Singularity is an assessment that I did at TAFE. It was made for a poster to promote the Fisher's Gig festival, originally made A3 size (below).

I decided to animate it later on in After Effects - this wasn't done for the assessment.

Like many of the pictures I've created along these same graphical styles, it uses Photoshop's 3D tools, if you take an image and turn it into a 3D depth map, turn that into an unlit texture, render it and then turn it into a smart object you will end up with an effect similar to this one here.
