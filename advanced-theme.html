T U R B U L E N C E


This is an advanced variant of the theme, it features less customisation than the advanced version.



Install Turbulence Advanced

Features

  • 1-6 columns (depending on your screen size)
  • Infinite scroll
  • A fully dynamic layout that will conform to many screen sizes
  • Parallax on scroll with the banner image
  • Posts can be any colour you like
  • Smooth scrolling that can be toggled on or off
  • Change fonts of the header, posts and navigation
  • 5 custom links
  • 12 variations for avatar styles
  • 4 variations for header text styles
  • Upload banner images either by uploading them from your computer or entering a URL to the image you wish to use
  • 17 different social media icons
  • custom CSS
  • 11 photo photo hover overlay patterns.
  • 4 post texture variations.
  • Change the RGB value to whatever colour you like for the photo hover effects on photo hover style 2 and 3. By default it is set to 0, 0, 0
  • Change the drop shadow colour of posts to whatever you like, the default colour is set to 0, 0, 0
  • Change the size and height of your logo image on photo posts.
  • change the height of the navigation bar to whatever you like, by default itâ€™s set to 60.
  • Add your hit-counter without having to go into the code.
  • Animations when the page is refreshed and when the page is scrolled.
  • 3 different photo post hover effect variations
  • More features to come in later updates
